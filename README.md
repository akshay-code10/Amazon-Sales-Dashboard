# 📊 Excel Data Analysis Project – Sales Dashboard

## 📌 Project Overview
This project focuses on analyzing e-commerce sales data using Microsoft Excel to generate actionable business insights and build an interactive dashboard for decision-making.

---

## 🎯 Objectives
- Clean and prepare raw sales data  
- Analyze sales performance across regions and categories  
- Identify trends, patterns, and key business insights  
- Build an interactive dashboard for data exploration  

---

## 🛠️ Tools & Techniques Used
- Microsoft Excel  
- Pivot Tables  
- Data Cleaning  
- Data Visualization (Charts & Graphs)  
- Slicers & Filters  
- Conditional Formatting  

---

## ⚙️ Steps Performed
### 1. Data Cleaning
- Handled missing values  
- Removed duplicates  
- Structured raw dataset  

### 2. Data Analysis
- Created Pivot Tables for summarization  
- Analyzed sales by region, category, and time  

### 3. Dashboard Creation
- Designed interactive dashboard using slicers  
- Added filters for dynamic analysis  
- Visualized trends using charts  

---

## 📊 Dashboard Features
- Sales by Region  
- Product Category Performance  
- Monthly Sales Trends  
- Cancellation Rate Analysis  
- Interactive Filters (Slicers)  

---

## 📈 Key Insights
- North region generated the highest sales  
- Electronics is the top-performing category  
- Cancellation rate is ~30%, indicating improvement scope  
- Top 3 regions contribute majority of revenue  

---

## 💡 Business Impact
- Helps identify high-performing regions and products  
- Supports better decision-making  
- Improves sales strategy and operational efficiency
